 <h2>レポート</h2>
 <p>「出所原則」で検索し、アーカイブズに関する論文をまとめた。</p>

 <h3><a href="https://www.jstage.jst.go.jp/article/archivalscience/11/0/11_73/_pdf/-char/ja" target="_blank">「アーカイブズと動的映像アーカイブ：近くて遠い隣人？」</a></h3>
 <p>著者：児玉優子</p>
 <p>閲覧：2025年1月19日</p>
 <p>　動的映像アーカイブがどのように発達してきたのか、アーカイブズ(学)においてこれまで動的映像資料がどのように扱われてきたかを振り返り、両者のギャップを埋め、今後の両者のあり方を論じている。動的映像とは、「記録方法や、画像を最初にまたは後に固定したフィルム、テープ、ディスクなどの支持体の性質に関わりなく、支持体に記録された一連の画像であり、音声を伴うことも伴わないこともあり、映写された時に運動の印象を与え、公衆へのコミュニケーションや頒布を意図したもの、または記録の目的で作成されるもの」である。また、動的映像アーカイブとは「動的映像資料および関連資料等を収集・管理・保存し、普及させることによって、コレクションへのアクセスを提供する権限を持つ組織または組織の一部」である。日本のアーカイブズにおいて、動的映像資料に関する取り組みはまだ不十分であり、図書館情報学の分野でより多く取り組まれてきた。児玉は、アーカイブズと動的映像アーカイブが密接にかかわるために、アーキビストの教育・研修の場で、紙以外のメディアの保存と取り扱いの知識を習得する機会が必要であると主張している。また、横のつながり、ネットワークを築くことで、互いに蓄積してきた知識と経験を共有し、資源や労力の無駄の回避することができると主張した。</p>

 <h3><a href="https://www.jstage.jst.go.jp/article/jcul/92/0/92_45/_pdf/-char/ja" target="_blank">「地域史料の情報公開に向けた近世文書目録のデータベース化について−山口大学図書館所蔵林家文書を事例として−」</a></h3>
 <p>著者：木越みち</p>
 <p>閲覧：2025年1月19日</p>
 <p>　山口大学図書館における林家文書のデータベース化を例として、図書館OB及び教員との連携、そしてその整理作業において試みた近年のアーカイブズ学の研究成果を反映させた目録データの工夫について、課題と成果及びその意義について述べている。書誌からでは個々の史料の詳細が分からず、目録からは史料の所在情報が特定できないという状態から再整理と入力作業を行い、分類項目を新たに設定するなどして、林家文書目録データベースを平成22年から平成23年春にかけて公開するに至った。課題として、人材育成についての問題点が挙げられている。林家文書目録データベース作成作業の中心はあくまでもOBであり、現職の図書館員はまだ十分なスキルを身につけたとは言えない。また、目録を担当した職員による自主的な勉強会は継続しているものの、目録作成に必要な知識を得るには、専門家である教員との連携が欠かせないと主張した。</p>

 <h3><a href="https://www.jstage.jst.go.jp/article/rmsj/49/0/49_KJ00003907025/_pdf/-char/ja" target="_blank">「オーストラリア連邦政府のレコードキーピング：リテンション・スケジュールと記録処分規定の比較を通して」</a></h3>
 <p>著者：坂口貴弘</p>
 <p>閲覧：2025年1月19日</p>
 <p>　記録連続体論の影響を受けたオーストラリア連邦政府におけるレコードキーピング・システムを例に、「記録連続体」概念が実務ではどのように適用されるか検討を行った論文である。記録連続体とは、レコードキーピングに関するシステムを設計する段階から、記録史料の段階までのすべてを総合的にとらえようとする概念のことである。また、レコードキーピングとは、業務の完全かつ正確で信頼できる証拠を、記録情報の形で作成し、維持することであり、これまでレコード・マネジメントと・マネジメントの呼ばれてきた領域と、これまで記録史料管理と呼ばれてきた領域の両方を包含する概念である。この論文では記録処分システム、記録分類の枠組み、作成プロセス、保存期間設定の基準を、アメリカとオーストラリア両国の国立公文書館における事例を中心とした比較を行っている。オーストラリアにおけるレコードキーピングについて、出所原則を解釈しなおして、記録分類の枠組みが組織別分類から機能別の分類へとシフトしてきているという点、記録の役割について、組織の壁を超えた社会・地域という観点を設定するなど、多元的にとらえていこうという認識が生まれてきている点が特徴として挙げられる。</p>

 <h3><a href="https://www.jstage.jst.go.jp/article/jcul/77/0/77_1213/_pdf/-char/ja" target="_blank">「アーカイブ資料整理への1つの試み-同志社大学所蔵田中稲城文書・竹林熊彦文書の場合-」</a></h3>
 <p>著者：井上真琴、小川千代子</p>
 <p>閲覧：2025年1月19日</p>
 <p>　アーカイブ資料整理の企画・発注側の担当者の井上と、業務指導コンサルタントの小川によって執筆されている。同志社大学において行われた田中稲城文書と，その母体コレクションである竹林熊彦文書の2年にわたる整理作業を事例に，図書館としてどのようにアーカイブ資料と向き合い、アウトソーシングを利用して整理し、公開に至ったかを報告している。同志社大学にアーカイブ資料の整理作業を担当できる人材はおらず、一から育成する余裕もない。また、アーカイブ科学の国際標準を知るコンサルタントの指導がなければ事業の遂行は難しいが、雇用は認められていないという状況の中、同志社大は次のような企画立案をした。<br>
    　整理作業はアウトソーシング業者に依頼し、アーカイブ科学のコンサルタントは業者側が探す。また、整理作業者と併せて業者側が契約する。紀伊國屋書店が業者側として参加し、この整理業務はほぼ当初の予定通りに完遂し、成功したと言える。アーカイブ資料整理はその分野についての深い理解と研究が不可欠であるが、同志社大の事例においては、コンサルタントと紀伊國屋書店の協議により、その都度作業進捗のために必要な調査研究であるか否かについての検討を行い、適切な情報の取捨選択が行われた。また、契約による業務ということで，作業に専念し、業務を推進したことが、ほぼ当初予定通りに整理作業が進捗した理由であると述べられた。</p>

 <h3><a href="https://www.jstage.jst.go.jp/article/ongakugaku/66/2/66_97/_pdf/-char/ja" target="_blank">「グレン・グールドの未整理・未公開資料に接して-一般の研究者はアーカイブの準備ができるのか-」</a></h3>
 <p>著者：宮澤淳一</p>
 <p>閲覧：2025年1月19日</p>
 <p>「一般の研究者はアーカイブの準備ができるのか」という問いに対し、カナダ・トロントでの宮澤の実体験から考察している。宮澤は「少なくともアーカイブ化の予備的な作業（仮の目録化と資料の保全）への関与はできる」と結論づけた。アーカイブ収容以前の未整理資料と出会い、その整理に関与するときに一般の研究者がとるべき望ましい行動について、宮澤は以下の10点のポイントを述べている。</p>
 <ol>
   <li>閲覧・整理・複写・利用・借用について、資料提供者・所有者との覚書を交わす。</li>
   <li>早めにプロのアーキビストと連絡をとる（助言や連携を求める）。</li>
   <li>複写や仮の目録化では、資料の取捨選択はしない（迷っている間に作業はできる）。</li>
   <li>可能な限り高画質で数通りの複写方法（デジタル化）を実行する。</li>
   <li>複写したデジタルデータのバックアップを万全にする。</li>
   <li>簡略化を工夫しつつも、第三者にもわかる形の目録を作る（ただし、標準的な目録規則をあらかじめ学んでおくのが望ましい）。</li>
   <li>資料提供者・所有者には資料に関して多くを何度も尋ね、記録を残す。</li>
   <li>資料提供者・所有者の資料評価に関する否定的な意見は、真に受けない。</li>
   <li>資料提供者・所有者には「ほかにないか」と尋ねる（尋ねたことで初めて出てくる資料もある）。</li>
   <li>資料は個人的に譲渡を受けない（資料の保全のためであれば、記録を残して預かるにとどめる）。</li>
 </ol>
   
 <h3><a href="https://www.jstage.jst.go.jp/article/jcul/78/0/78_1234/_pdf/-char/ja" target="_blank">「名古屋大学附属図書館における記録史料の整理・保存・公開」</a></h3>
 <p>著者：秋山晶則</p>
 <p>閲覧：2025年1月19日</p>
 <p>名古屋大学附属図書館所蔵高木家文書を事例に、近年の動向もふまえながら、記録史料をめぐる図書館の対応について報告している。記録史料は、村や家などの組織体を母体に作成・伝達・蓄積され、当時の社会関係やその組織構造を反映した個性的な史料「群」として存在しているため、史料相互の連関性，有機的な結びつきが失われてしまわないよう，整理原則としての定式が鍛えあげられてきた。</p>
 <ol>
   <li>出所原則（史料群の混在禁止）</li>
   <li>原秩序尊重（史料排列等の情報保全）</li>
   <li>原形保存（包み方など現状の保全）</li>
   <li>段階的整理（分析レベルを順次深化させ，史料群の構造認識に至る方法）</li>
 </ol>
 <p>である。一方でその定式がすべて完璧に守られるかといわれると、そうではなく、記録資料の整理の段階で多くの情報が失われるという反省点がある。この論文では、地域連携、保存と修復、公開のあリ方(閲覧と展示、デジタル化)の3つのセクションに分け、記録史料の情報資源化について解説している。また、段階的に取り組むことが重要であると主張している。</p>


 <h3><a href="https://www.jstage.jst.go.jp/article/jsai/31/0/31_30/_pdf/-char/ja" target="_blank">「地方公文書館における館と機能の議論について：1990年頃と2020年の引継移管を比較して」</a></h3>
 <p>著者：清水惠枝</p>
 <p>閲覧：2025年1月19日</p>
 <p>公文書とその管理に関する法律について、公文書管理の例規を用いて1990年頃と2020年の引継移管の方法を比較し、1990年頃よりも2020年の方が引継移管の方法が豊富になっており、公文書管理における公文書館の関与が明確で、専門部署としての位置付けも自治体組織内で認識されていることを論じている。また、民主的な自治のために重要な制度として公文書管理が位置づけられ、運用をしながら鍛えられていくべきだと主張している。<br>
    1990年頃における引継移管の課題は2020年の引継移管において大方解決されたと言える。一方で、どのような例規レベルであればいいのかという問題について、なるべく属人的な対応は減らすべきとし、公文書管理の質を担保するため、ルールの明文化が必要であるとした。そして、規定を作る過程には多くの人が意思決定に関わること、公文書管理を住民の行政参加に関する制度と関連づけをすること、ルールに当てはまらない新しい事案に対し、それを新た取り込み、公文書管理を育てていくことなどが重要であるとした。さらに、専門職アーキビストの身分に触れ、認証制度が行政組織内の人事評価と連動することが必要であると主張した。</p>
